# 매뉴얼 감사 매트릭스 v0.1

작성일: 2026-07-05  
대상: v1.0 원본을 직접 수정하지 않고 작업본에서 감사한다.

## 1. 감사 판정

| 판정 | 의미 |
| --- | --- |
| OK | v1.0 내용이 실무 실행 가능한 수준 |
| PARTIAL | 항목은 있으나 책임자, 수치 기준, 증빙, 서식, Gate가 부족 |
| GAP | 실무자가 따라 할 절차가 없음 |
| UPDATE | 법령·정책·금융 관행 기준시점 재검증 필요 |
| DUPLICATE | 이미 충분한데 중복 보강 위험 |

## 2. Phase별 감사 질문

| Phase | 감사 질문 |
| --- | --- |
| Phase 1 사업기획/토지 발굴 | 사업 컨셉, 수요, 분양가, PF 가능성, 토지 소유권, 개발업 컴플라이언스 초기 확인이 있는가 |
| Phase 2 토지 확보/DD | 등기, 권리, 유치권, 명도, 지상권, 접도, 기반시설, 토양오염, 문화재, 데이터룸 checklist가 있는가 |
| Phase 3 사업타당성 | 자기자본, 브리지론, 본PF, 금리, 공사비, 세무·회계·SPC, 미분양 stress test가 수치화되어 있는가 |
| Phase 4 인허가 | 심의, 교통영향평가, 기부채납, 주민·지자체 이슈, 재상정 buffer, 문서증거 보존이 있는가 |
| Phase 5 PF 자금조달 | 2027년 이후 자기자본비율 전환, 위험가중치, 충당금, 대출취급 요건, 보증료 우대, PF 통합정보시스템 대응이 있는가 |
| Phase 6 시공 관리 | 시공사 실사, 책임준공, 공사비 증액, 물가변동, 하도급, 보험·보증, 재난·안전 trigger가 있는가 |
| Phase 7 분양/마케팅 | 분양보증, 표시광고, 분양대행, 개인정보, 계약 해제, 미분양 대응, 가격 조정 승인 기준이 있는가 |
| Phase 8 준공/정산 | 사용승인, PF 상환, 하자보수, 잔여재고 exit, 세무 마감, 담보 해제, 정산보고, 자료보존이 있는가 |

## 3. Cross-Cutting 감사 축

| 축 | 확인 항목 |
| --- | --- |
| 원본/버전관리 | 원본 보존, 작업본, 변경이력, diff, 승인 로그 |
| Gate | 통과 기준, stop condition, 예외승인권자, 증빙 폴더 |
| 책임자/RACI | 담당, 검토, 승인, 외부전문가, 재검토 주기 |
| 세무·회계·SPC | 취득세, 부가세, 법인세, 배당, 이자비용, 자금집행, 특수관계 |
| 데이터룸·개인정보 | 권한, 원본성, 보존기간, 개인정보 최소수집, 유출 대응 |
| 계약/조달 | 시공, 신탁, PM, 분양대행, 금융, 설계, 감리, default/step-in |
| 교육훈련 | 신규 담당자 onboarding, Gate drill, red-team 질문, failure replay |
| 준공 후 | warranty, reserve, 미분양, 정산, 문서 인계 |
